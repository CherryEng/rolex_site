# Rolex商品紹介サイト

## ページ構成
- index.html (商品一覧) 
- page01.html (詳細ページ)

---

## リンク
### 詳細ページへのリンク
- ヘッダーナビゲーション内のカルーセル
- ファーストビューの"View recommended product"
- 商品一覧の画像

### トップページへのリンク
- ヘッダー、フッターの"Rolex watches"

その他リンクはhref="#"に設定

---

## 機能
- ハンバーガーメニューによるナビゲーション展開
- スクロール方向によるヘッダー表示切り替え
- タブメニューによるコンテンツ表示切替
- swiperによるカルーセルスライダー
- スクロール時のフェードインアニメーション
- collectionセクションの慣性スクロールと表示モード切替
- スクロール量に応じた連番画像でのアニメーション表現 
- 連番画像による時計回転表現 (自動回転 ON・OFF切替 )

---

## モジュール
### scss
共通とページごとにフォルダを分けて、ページ固有scssはセクション単位で分割

### JS
ES6modulesの機能で、共通ファイルと各ページの各機能ごとに分割
